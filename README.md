PROJETO: MONITOR DE IMPACTO AMBIENTAL (ECOTRACK) – 2026

Desenvolvimento de um dashboard interativo utilizando Python e Streamlit para cálculo e monitoramento de pegada de carbono diária.
Visualização de dados dinâmica com gráficos para análise de consumo e feedback instantâneo ao usuário sobre metas de sustentabilidade.
Versionamento e documentação do código no GitHub, incluindo guias de instalação e requisitos.
