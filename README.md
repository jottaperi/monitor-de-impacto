ROJETO: MONITOR DE IMPACTO AMBIENTAL (ECOTRACK) – 2026Desenvolvimento de um dashboard interativo utilizando Python e Streamlit para cálculo e monitoramento de pegada de carbono diária.Implementação de lógica matemática para conversão de atividades (transporte, energia e alimentação) em emissões de $CO_2$, baseada em fatores de emissão reais.Visualização de dados dinâmica com gráficos para análise de consumo e feedback instantâneo ao usuário sobre metas de sustentabilidade.Versionamento e documentação do código no GitHub, incluindo guias de instalação e requisitos.
